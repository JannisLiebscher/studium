==========================
JERM - Just an ER Modeler
==========================

SYSTEMANFORDERUNGEN:
--------------------
Betriebssystem: Windows 98 oder höher, Linux, Mac OS X
CPU: 733 MHz
Speicher: 256 MB RAM
Auflösung: 800x600

INSTALLATION:
--------------------
Falls Sie die Installer-Version runtergeladen haben starten Sie das 
Installationsprogramm durch einen Doppelklick auf die Setup.exe und folgen 
Sie dem Installationsverlauf.

Beim einfachen Zip-Paket entpacken Sie bitte das Paket einfach in ein 
Verzeichnis Ihrer Wahl und folgen Sie den Anweisungen unter dem Abschnitt 
START.

START:
--------------------
Um das Programm zu starten sollte ein Doppelklick auf das Desktop Icon genügen. 
Ansonsten müssen Sie das Programm manuell starten, indem Sie auf der 
Kommandozeile ins Programmverzeichnis wechseln und dort eingeben:

java -jar "JERM.jar" 

Im Programmordner befindet sich auch ein Beispiel-Diagramm, das die Möglichkeiten
des Programms aufzeigt.
